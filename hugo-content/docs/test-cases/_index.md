# GOTM Test Cases

Example simulations demonstrating GOTM capabilities.

**Repository:** https://github.com/gotm-model/cases

## Quick Start

```bash
git clone --recurse-submodules https://github.com/gotm-model/cases.git
cd cases/<case_name>
gotm
```

Output: `result.nc` (NetCDF file with model results)

## Categories

- [Idealized](idealized/) - Process studies and fundamentals
- [Realistic](realistic/) - Real-world scenarios with observations
