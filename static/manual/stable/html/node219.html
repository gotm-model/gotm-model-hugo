Fortran: Module Interface seagrass -- sea grass dynamics (Source File: seagrass.F90)


INTERFACE: 

    module seagrass
DESCRIPTION:

In this module, seagrass canopies are treated as Lagrangian tracers, which either advect passively with the horizontal current speed or rest at their excursion limits and thus exert friction on the mean flow, see Verduin and Backhaus (2000). Turbulence generation due to seagrass friction is possible, see namelist file seagrass.nml. The extra production term in the balance of TKE, (150), is included as described in section 4.7.21.
