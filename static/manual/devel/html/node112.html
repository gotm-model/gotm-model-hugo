Read heat flux data, interpolate in time


INTERFACE: 

   subroutine read_heat_flux(jul,secs,heat)
DESCRIPTION:

For calc_fluxes=.false., this routine reads in the sum of sensible, latent and long-wave back-radiation flux in Wm from heatflux_file and interpolate in time.


USES: 

   IMPLICIT NONE
INPUT PARAMETERS: 
   integer, intent(in)                 :: jul,secs
OUTPUT PARAMETERS: 
   REALTYPE, intent(out)               :: heat
