In case you want to add your GOTM publication to the list, please provide the Bibtex information to [us](mailto:hans.burchard@iow.de). 
We will then add it to the list and update the website accordingly.

The GOTM publication list tries to cover mostly publications with GOTM as a water-column model, excluding mere 3D model applications just using GOTM as a turbulence model. The list also contains a number of publications where GOTM was used in lake mode. The lake code is no longer maintained (as of around 2018) and not part of the main repository. The old version is still usable and is available as is. 
An alternative for lake modelling is [simstrat](https://www.eawag.ch/en/department/surf/projects/simstrat/).
